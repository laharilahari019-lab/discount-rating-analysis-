# Discount vs Rating Causation Analysis

## Objective
Analyze whether higher discounts improve customer satisfaction or just increase review volume.

## Tools Used
- Python (Pandas, Seaborn)
- Google Colab
- Power BI

## Key Insights
- Discounts significantly increase review volume
- Ratings remain stable when controlling for review count
- Indicates volume-driven rating bias, not actual satisfaction improvement
